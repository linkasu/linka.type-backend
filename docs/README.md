# Linka Type Backend Docs

Start here for architecture and specs. This repo is split into three services and uses Yandex Cloud serverless containers + YDB with Firebase compatibility.

## Index
- `architecture.md` - system overview and components
- `data-model.md` - YDB tables, keys, and field semantics
- `api.md` - REST + realtime endpoint contracts
- `migration.md` - Firebase compatibility, dual-write, and sync strategy
- `realtime.md` - change feed, cursors, long polling, WebSocket
- `deployment-yc.md` - Yandex Cloud deployment notes
- `local-dev.md` - local development notes
