LINKa — поддержка общения

Помогаем людям с нарушениями речи говорить через понятный текст.

LINKa Type API хранит пользовательские фразы, категории и быстрые ответы, поддерживает онбординг и дает realtime-обновления. Это сердце приложения LINKa на linka.su.

Попробовать API Документация клиента
REST + WebSocket Token Auth YDB + Sync

Что умеет платформа

  • Категории и фразы пользователя с сохранением истории.
  • Быстрые реплики, состояния онбординга и импорт глобальных наборов.
  • Realtime синхронизация через long-polling и WebSocket.
Стек: Go + YDB + Firebase
Функциональность

API для приложений альтернативной коммуникации

LINKa Type помогает сохранять персональные фразы и быстро собирать ответы. API аккуратно мигрирует данные из Firebase и поддерживает реалтайм, чтобы клиенты всегда были синхронизированы.

01

Категории и фразы

CRUD для категорий и утверждений, с историей обновлений и метаданными.

02

Быстрые ответы

Сохраняйте короткие реплики и подключайте их к интерфейсу клавиатуры.

03

Онбординг

Готовые вопросы и генерация стартовых фраз для первых экранов приложения.

04

Realtime

Получайте изменения списков и фраз через /v1/changes и /v1/stream.

Как подключиться

  1. Получите токен через POST /v1/auth (email + пароль).
  2. Выберите базовый URL (Gateway или контейнер core-api).
  3. Синхронизируйте категории, фразы и быстрые ответы.

Подробности и примеры запросов — в гайде разработчика.

Realtime в двух вариантах

Long polling

GET /v1/changes?cursor=...

WebSocket

WS /v1/stream?cursor=...

Оба формата возвращают курсор и список изменений.

Подсказка

Базовый URL по умолчанию — текущий домен, если фронтенд и API развернуты вместе.

Безопасность

Токен хранится только в памяти браузера и не отправляется на сторонние сервисы.
