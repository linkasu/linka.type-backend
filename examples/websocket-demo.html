Тестирование API

Используйте эти команды curl для тестирования (замените YOUR_TOKEN на ваш токен):

# Создать категорию
curl -X POST http://localhost:8081/api/categories \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"title": "Тестовая категория"}'

# Обновить категорию
curl -X PUT http://localhost:8081/api/categories/CATEGORY_ID \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"title": "Обновленная категория"}'

# Удалить категорию
curl -X DELETE http://localhost:8081/api/categories/CATEGORY_ID \
  -H "Authorization: Bearer YOUR_TOKEN"

# Создать statement
curl -X POST http://localhost:8081/api/statements \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"title": "Тестовый statement", "categoryId": "CATEGORY_ID"}'

# Обновить statement
curl -X PUT http://localhost:8081/api/statements/STATEMENT_ID \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"title": "Обновленный statement", "categoryId": "CATEGORY_ID"}'

# Удалить statement
curl -X DELETE http://localhost:8081/api/statements/STATEMENT_ID \
  -H "Authorization: Bearer YOUR_TOKEN"
